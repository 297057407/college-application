# college-application
## 如何运行？
### 1.安装 Node.js 和 npm：
如果您的电脑上尚未安装 Node.js 和 npm，您需要首先安装它们。您可以从 Node.js 官方网站下载16及以上版本的 Node.js，npm 会随着 Node.js 一同安装。

### 2.进入项目目录：
```sh
cd <项目目录>
```
### 3.安装依赖：
```sh
npm install
```
### 4.启动项目
```sh
npm run dev
```


### 编译和压缩以用于生产环境
```sh
npm run build
```
### 使用ESLint进行代码检查(https://eslint.org/)

```sh
npm run lint
```
